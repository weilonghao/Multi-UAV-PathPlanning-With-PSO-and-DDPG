# Multi-UAV-PathPlanning-With-PSO-and-DDPG
这个项目是一个多无人机（UAV）路径规划系统，结合了两种算法： 粒子群优化算法（PSO）： 在 PSO_3_drones 和 PSO_5_drones 文件夹中实现 用于优化多无人机的巡逻路径 考虑了飞行距离、时间、负载均衡和目标生存概率等因素 强化学习算法（DDPG）： 在 RL_3_drones 文件夹中实现 使用深度确定性策略梯度（DDPG）算法 学习如何最优地分配目标和规划路径 项目特点： 支持3架和5架无人机的场景 包含目标聚类功能，减少需要访问的目标点数量 多目标优化：同时考虑距离、时间、负载均衡和生存概率 可视化路径规划和训练曲线
